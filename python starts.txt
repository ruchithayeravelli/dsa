✅ Simple definition:
A variable in Python is a name that stores a value in memory so it can be used later in a program.

Types of Variables in Python:
1. Local Variable

A local variable is created inside a function and can only be used inside that function.

Example:

def show():
    **x = 10   # local variable**

    **print(x)**




show()
2. Global Variable

A global variable is created outside all functions and can be used anywhere in the program.

Example:

x = 20   # global variable

def show():
    **print(x)**
show()
3. Instance Variable

Used in classes and belongs to objects.

Example:

class Student:
    **def \_\_init\_\_(self, name):**
        **self.name = name   # instance variable**
s1 = Student("Ravi")
print(s1.name)

Each object has its own copy of the variable.
4. Class Variable

A class variable is shared by all objects of the class.

Example:

class Student:
    **school = "ABC School"   # class variable**




s1 = Student()
s2 = Student()

print(s1.school)
print(s2.school)

Both objects access the same variable.
